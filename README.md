# e-commerce-midgard
E-commerce simple
Midgard é um projeto de e-commerce moderno desenvolvido com Next.js e Tailwind CSS, focado em performance, escalabilidade e uma experiência de usuário fluida.
O projeto utiliza uma arquitetura limpa e componentes reutilizáveis, com foco em boas práticas de front-end e preparação para integração com um banco de dados simples para gerenciamento de produtos, usuários e pedidos.

Este repositório tem como objetivo servir tanto como um projeto real quanto como estudo e evolução contínua em desenvolvimento web moderno.

Tecnologias principais:

Next.js

React

Tailwind CSS
